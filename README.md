📚 Sobre o Projeto

Este projeto foi desenvolvido durante o ensino médio como parte da minha jornada inicial no desenvolvimento web. Ele marcou meus primeiros contatos práticos com HTML, CSS e a construção de interfaces, sendo um passo importante no meu aprendizado na área.

Após sua finalização, o projeto permaneceu cerca de 2 anos sem receber atualizações. Hoje, ele representa não apenas um registro da minha evolução como desenvolvedor, mas também uma base valiosa para revisitar conceitos, aplicar melhorias e continuar aprendendo.

🚀 Este repositório demonstra o quanto evoluí ao longo do tempo e reforça meu interesse contínuo em aprimorar minhas habilidades.
